# Fintech Review Analytics – Ethiopian Banks Mobile Apps

## Overview

This project provides a complete data engineering and analytics pipeline for scraping, cleaning, analysing, and visualising Google Play Store reviews of three major Ethiopian banking apps:

- Commercial Bank of Ethiopia (CBE)  
- Bank of Abyssinia (BOA)  
- Dashen Bank

The pipeline transforms raw user feedback into actionable business insights: sentiment classification (positive/negative/neutral), thematic extraction (e.g., performance, authentication, features), and stakeholder‑ready visualisations.

This work was completed as Week 2 Challenge of the *10 Academy Artificial Intelligence Mastery* programme.
